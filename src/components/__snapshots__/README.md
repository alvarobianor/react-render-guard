# Snapshots

The `.snap` files in this folder are automatically generated by Vitest
when the snapshot tests (`snapshot.test.tsx`) run.

To update snapshots after an **intentional** UI change, run:

```bash
npm test -- --update-snapshots
# or
vitest --update-snapshots
```

> **Do not edit `.snap` files manually** — they are the source of truth
> used to detect unintentional structural or visual regressions in components.
